Compute Example

Source file: .github/examples/compute.dcrt

This MIPS target program demonstrates basic arithmetic operations: loading immediate values, addition, subtraction, and passing results between registers. It initializes r1 = 12, r2 = 1, r3 = 100, then computes r4 = r1 + r2 and r3 = r3 - r2.

Features Demonstrated

  • li for loading immediate values
  • add for register addition
  • sub for register subtraction
  • MIPS register usage conventions

Build

cargo run -- compile-arch .github/examples/compute.dcrt --out-file build/compute_mips.bin

The output is a raw binary containing the encoded MIPS instructions.

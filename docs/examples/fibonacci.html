Fibonacci Example

Source file: .github/examples/fibonacci.dcrt

This ARM Cortex-M program computes the Fibonacci sequence to 20 iterations. It demonstrates a loop with conditional branching, register tracking of multiple values, and ARM Thumb instruction encoding.

Features Demonstrated

  • mov for register initialization
  • add for Fibonacci addition
  • cmp and beq for loop condition testing
  • b for unconditional branching back to loop start
  • ARM Thumb register conventions

Build

cargo run -- compile-armcm .github/examples/fibonacci.dcrt --out-file build/fibonacci.bin

The output is a raw binary suitable for flashing to ARM Cortex-M microcontrollers.

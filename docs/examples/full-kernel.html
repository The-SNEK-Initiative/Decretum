BIOS Kernel Example

Source file: .github/examples/bios_kernel.dcrt

This sample is a BIOS bootable Decretum kernel for target bios16. It demonstrates a full interactive menu with status display, memory detection, keyboard input, disk operations, beep, echo, reboot, and halt commands. The kernel runs on bare metal x86 hardware in real mode.

Features Demonstrated

  • builtin.print and builtin.println for text output
  • builtin.get_mem_kb to detect conventional memory size via INT 12h
  • builtin.get_cursor and builtin.set_cursor for cursor control
  • builtin.memcpy and builtin.memset for memory operations
  • builtin.wait_key for keyboard input
  • builtin.disk_reset for disk controller reset
  • builtin.beep for speaker output
  • builtin.reboot and builtin.halt for system control
  • if and while control flow constructs
  • Event dispatch with emit
  • Comparison based menu dispatch with cmp and je

Build

cargo run -- compile-bootimg .github/examples/bios_kernel.dcrt --out-file build/bios_kernel.img

Output Files

  • build/bios_kernel.img (boot sector plus stage2 kernel)
  • build/bios_kernel.kernel.bin (stage2 payload only)

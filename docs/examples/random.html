Random Numbers Example

Source file: .github/examples/random.dcrt

This portable bytecode program demonstrates the DCRNG random number generator. It generates 10 random numbers in the range 0 to 99 using random_range and prints them. The DCRNG is a 320 bit ARX PRNG with a Weyl sequence for high quality pseudo random output.

Features Demonstrated

  • random_range for bounded random number generation
  • print and print_u64 for text output
  • println for line breaks
  • inc for counter increment in a loop
  • exit for clean program termination
  • Portable bytecode target with DCRNG

Build

cargo run -- compile-bytecode .github/examples/random.dcrt --out-file build/random.dcb

The output is a .dcb bytecode file that can be run through the IDE or compiled to a PE executable.

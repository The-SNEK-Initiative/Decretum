String Ops Example

Source file: .github/examples/string_ops.dcrt

This portable bytecode program demonstrates the string manipulation operations available in Decretum. It concatenates strings, measures length, finds substrings, and converts integers to strings using the portable bytecode runtime's string subsystem.

Features Demonstrated

  • str_alloc for buffer allocation
  • str_cat for string concatenation
  • str_len for length measurement
  • str_find for substring search
  • int_to_str for integer formatting
  • print_data for buffer output
  • Portable bytecode string subsystem

Build

cargo run -- compile-bytecode .github/examples/string_ops.dcrt --out-file build/string_ops.dcb

The output is a .dcb bytecode file that can be run through the IDE or compiled to a PE executable.

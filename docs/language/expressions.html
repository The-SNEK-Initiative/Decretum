Instruction Model

Inside event and proc blocks, Decretum accepts instruction lines directly. Most operations use two operand form (dest, src). Some high level ops use three or four operands.

Arithmetic

InstructionFormDescription
movdest, srcCopy source value to destination
adddest, srcdest = dest + src
subdest, srcdest = dest - src
muldest, srcdest = dest * src
divdest, srcdest = dest / src
moddest, srcdest = dest % src
incdestdest = dest + 1
decdestdest = dest - 1

Bitwise

InstructionFormDescription
anddest, srcdest = dest & src
ordest, srcdest = dest | src
xordest, srcdest = dest ^ src
notdest, srcdest = ~src
shldest, srcdest = dest << src
shrdest, srcdest = dest >> src

Comparison and Flow

InstructionFormDescription
cmpa, bCompare a against b, sets signed and unsigned flags
jmplabelUnconditional jump
je / jnelabelJump if equal or not equal
jl / jlelabelJump if less or less or equal (signed)
jg / jgelabelJump if greater or greater or equal (signed)
jb / jbelabelJump if below or below or equal (unsigned)
ja / jaelabelJump if above or above or equal (unsigned)
calllabelCall procedure by label
retReturn from procedure

High Level Operations

These operations extend the core instruction set with string handling, math utilities, and runtime services. They are available in the portable bytecode path and may not be available in direct machine code backends.

InstructionFormDescription
absdest, srcdest = absolute value of src
mindest, a, bdest = smaller of a and b
maxdest, a, bdest = larger of a and b
clampdest, val, lo, hidest = clamped value between lo and hi
powdest, base, expdest = base raised to exp (wrapping)
time_msdestUnix timestamp in milliseconds
random_rangedest, lo, hiRandom integer in range lo to hi inclusive
assertcond, "msg"Halt if cond is zero, print message
int_to_strstr_var, valueConvert integer to string
str_to_intstr_var, destParse integer from string
str_catdst, a, bConcatenate strings a and b into dst
str_findstr, sub, destFind substring position or -1 if not found

Block Control Flow Constructs

These are compiled to compare and jump sequences at the source level. Both constructs support arbitrary nesting.

ConstructDescription
if cond ... else ... endifConditional execution with optional else branch
while cond ... endwhileLoop while condition is nonzero

Labels

.loop:
    dec rcx
    jnz .loop

Labels ending with a colon mark a position in the code. Local labels starting with a dot are scoped to their enclosing block to prevent name collisions across events and procedures.

Event Dispatch

emit irq_timer

The emit instruction dispatches directly to the named event handler. Events can pass control flow through shared data declarations.

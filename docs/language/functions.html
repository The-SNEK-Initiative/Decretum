Helper Functions

Portable and PE Runtime Helpers

These helpers are available in the portable bytecode and PE runtime paths. They provide I/O, timing, event dispatch, and self compilation services.

HelperDescription
print "text"Write string to stdout
printlnWrite newline
println "text"Write string plus newline
print_data labelWrite data declaration by label
print_u64 operandPrint unsigned 64 bit value
print_var operandPrint numeric value from register or memory
wait_inputRead one input byte before continuing
input varRead a line and parse an integer into var
exit codeExit process with code
sleep_ms nSleep for n milliseconds
emit nameCall event handler by name
read_file dest_str, pathRead file contents into string variable
write_file path, src_strWrite string contents to file
compile_decretum src, out, targetCompile Decretum source from a string variable

BIOS Builtins (bios16 target only)

These helpers are specific to the bios16 target and lower directly to 16 bit x86 machine code with BIOS interrupt calls.

HelperMeaning
builtin.print labelBIOS print using NUL terminated data label
builtin.print_u16 valueBIOS decimal print for 16 bit value
builtin.print_hex16 valueBIOS hexadecimal print for 16 bit value
builtin.wait_keyBIOS keyboard read
builtin.get_mem_kbRead conventional memory size in KB into AX
builtin.memcpy dst, src, countCopy byte ranges using rep movsb
builtin.memset dst, value, countFill byte ranges using rep stosb
builtin.set_video_mode modeSwitch BIOS video mode through INT 10h
builtin.get_cursorRead cursor row and column into DH and DL
builtin.rebootBIOS warm reboot through INT 19h
builtin.panic labelPrint message, disable interrupts, halt loop
builtin.haltEmit hlt instruction

Use builtin.* helpers in bios16 targets and standard instructions in portable or win64 targets.

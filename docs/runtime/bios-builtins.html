BIOS Builtins

These helpers are available in target bios16 programs and lower directly to 16 bit x86 machine code with BIOS interrupt calls. They provide console I/O, video control, keyboard input, memory operations, disk services, and system control.

HelperDescription
builtin.print labelPrint NUL terminated string at label using INT 10h teletype output
builtin.println labelPrint label string followed by CRLF
builtin.newlineEmit carriage return and linefeed
builtin.print_char imm8 or 'c'Print one character directly
builtin.print_u16 valuePrint unsigned 16 bit value in decimal
builtin.print_hex16 valuePrint unsigned 16 bit value in hexadecimal
builtin.clear_screenSwitch to text mode 03h and clear the screen
builtin.set_video_mode modeSet BIOS video mode through INT 10h AH=00h
builtin.wait_keyRead one key using BIOS INT 16h AH=00h, blocks until keypress
builtin.get_keyNon blocking keyboard check using INT 16h AH=01h
builtin.set_cursor row, colSet cursor position using INT 10h AH=02h
builtin.get_cursorRead cursor position using INT 10h AH=03h, row in DH, column in DL
builtin.set_text_attr imm8Set text attribute byte used by teletype output
builtin.get_mem_kbRead conventional memory size in KB into AX via INT 12h
builtin.memcpy dst, src, countCopy count bytes using rep movsb
builtin.memset dst, value, countFill memory with value using rep stosb
builtin.beepEmit bell character (0x07) through teletype output
builtin.disk_resetCall INT 13h AH=00h to reset disk system
builtin.rebootWarm reboot through INT 19h
builtin.haltEmit hlt instruction, halts the CPU until next interrupt
builtin.panic labelPrint message, newline, disable interrupts, enter infinite halt loop

All builtins assume the CPU is in real mode with BIOS interrupt services available. They are not available in UEFI, protected mode, or long mode targets.

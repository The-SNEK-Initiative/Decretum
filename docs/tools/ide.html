Decretum IDE

The IDE is a full featured code editor built with egui and eframe in Rust. It provides a complete development environment for Decretum programs with syntax highlighting, compilation, execution, and built in documentation.

Build and Run

cargo build --release --bin ide
cargo run --bin ide

Editor Features

  • Multi tab editor with independent undo and redo stacks (50 levels each)
  • Autosave with configurable interval
  • Line number gutter with error markers
  • Syntax highlighting for keywords, registers, builtins, strings, labels, comments, and error lines
  • Code snippets for portable hello world, BIOS skeleton, RISC-V CHERI, ARM Cortex-M, 6502, loop, and procedure templates
  • Find and replace with case sensitive matching, find next, replace next, and replace all
  • Line jump and file navigation

Panels

  • Explorer panel: Recursive workspace file browser that excludes git and build directories. Supports file open, rename, delete, and create with context menus.
  • Outline panel: Shows parsed symbols from the active file including events, procedures, target, entry point, and data declarations.
  • Inspector panel: Displays program summary with blocks, data declarations, and parse errors. Shows build history and provides an external command runner.
  • Reference panel: Built in documentation viewer with 13 pages covering overview, getting started, syntax, instruction model, helpers, execution model, constraints, BIOS builtins, backends, PE target, IDE usage, and examples.

Compilation and Execution

  • Compile targets include PE (.exe), bytecode (.dcb), BIOS16 (.img), RISC-V (.bin), and RISC-V CHERI (.bin)
  • Inline compiler output with stdout and stderr capture
  • Runtime execution of .dcb files through the embedded portable bytecode interpreter
  • Execution of .exe files as subprocesses
  • QEMU launch support for .img and .bin files
